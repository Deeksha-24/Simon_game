How to play the game

1. To start the game, Press any key from keyboard.

2. When then game starts, it will show you the tile you have to click on.

3. With each level, the sequence would become more complex and you have to remember all the colours in sequence from the start.

4. If you get the past sequence correct you will move to next level, otherwise the game stop.

Example:- You pressed any key from keyboard

Level 0: Computer's press- Red

You- Red

Level 1: Computer press- Blue

You- Red, Blue

Level 2: Computer- Green

You- Red,Blue,Green

Level 3: Computer- Red

You-Blue. (Mismatch)

Game Over. Play Now
